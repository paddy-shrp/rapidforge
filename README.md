# Installation

## Repository

```bash
git clone https://github.com/paddy-shrp/rapidforge.git
cd rapidforge
```

## Setup the Python Environment
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
pip install -e rapidforge_core
```
